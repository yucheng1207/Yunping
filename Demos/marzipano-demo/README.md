# 创建项目

## useful-cli

本项目使用[useful-cli](https://github.com/yucheng1207/useful-cli)创建。

## node 版本

v14.18.1

# 开发

## 开发调试

执行`yarn dev`命令可本地运行该项目，打开[http://localhost:3000](http://localhost:3000)可在线调试。

```bash
yarn dev
```

## 编译

执行`yarn build`可以编译项目，输出路径为`build`文件夹

```bash
yarn build
```
